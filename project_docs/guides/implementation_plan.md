# 구현 계획 (Implementation Plan)

## 개요
이 문서는 Vite 기반 React 프로젝트에서 빌드(`npm run build`) 후 결과물을 확인하거나 개발 서버를 실행하는 방법에 대한 안내 계획입니다.

## 제안된 내용
- **현상 분석**: `npm run build`는 코드를 최적화하여 `dist` 폴더에 정적 파일을 생성할 뿐, 서버를 띄우지 않습니다. 또한, `npm run dev`는 터미널에 종속된 프로세스이므로 터미널 종료 시 서버도 함께 중지됩니다.
- **해결 방안 1 (개발 서버 실행)**: 코드를 수정하며 화면을 보려면 `npm run dev` 명령어를 사용해야 합니다. (localhost:5173) 터미널 창은 계속 열어두어야 합니다.
- **해결 방안 2 (빌드된 파일 확인)**: 빌드된 프로덕션 결과물을 확인하려면 `npm run preview` 명령어를 사용합니다.
- **해결 방안 3 (편의성)**: 별도의 CMD 창 대신 에디터(VS Code 등)의 통합 터미널을 사용하면 화면 관리가 더 수월합니다.
