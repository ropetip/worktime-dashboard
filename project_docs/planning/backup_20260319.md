## 데이터베이스 구조(Supabase)
- **Table Name:** `schedules`
- **Columns:**
  - `id`: 고유 ID (UUID)
  - `date`: 날짜 (DATE, YYYY-MM-DD)
  - `name`: 이름 (TEXT)
  - `time`: 시간 코드 (TEXT, 예: '0800', '0900', 'OFF')
  - `reason`: 사유/메모 (TEXT)
  - `created_at`: 생성 일시
