# 📅 파트 근무 현황 대시보드 (React + Vite)

이 프로젝트는 파트원들의 근무 현황을 효율적으로 관리하고 시각화하기 위해 기존 JSP 시스템을 현대적인 React 기반의 SPA(Single Page Application)로 마이그레이션한 결과물입니다.

## ✨ 주요 기능
- **자동 로테이션**: 08시 출근 순번을 알고리즘으로 자동 배정합니다.
- **실시간 대시보드**: 18시 상주 인원 준수 여부를 즉시 계산하여 보여줍니다.
- **직관적인 캘린더**: 프리미엄 UI가 적용된 달력에서 팀원들의 일정을 한눈에 파악할 수 있습니다.
- **예외 일정 관리**: 휴가, 외근 등의 일정을 간편하게 추가/수정/삭제 가능합니다.

## 🚀 빠른 시작 (실행 방법)

1.  의존성 설치: `npm install`
2.  로컬 실행: `npm run dev`
3.  배포 파일 생성: `npm run build`

> **Note**: 본 프로젝트는 향후 Supabase 연동을 통해 실시간 DB 기능을 지원할 예정입니다.
